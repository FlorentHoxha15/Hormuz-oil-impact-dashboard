# 🌍 Oil Crisis Impact Dashboard (2026)

⚠️ What happens to the world if a key oil supply route is disrupted?

This mini data analyst project explores the potential global impact of a closure of the Strait of Hormuz using a scenario-based Power BI dashboard.

---

## 📊 Project Overview

The Strait of Hormuz is one of the most important oil transit routes in the world.  
In this project, I built a dashboard to simulate how different regions would be affected if oil supply is disrupted.

The dashboard allows users to:
- Switch between different crisis scenarios
- Compare regional impact levels
- Visualize the results using charts and a heatmap

---

## 🔍 Key Insight

Asia is by far the most vulnerable region, followed by Europe, due to high dependency on imported oil.

---

## 🛠 Tools Used

- Power BI (DAX, data visualization)
- Excel (data modeling)
- Scenario analysis


---

## 📈 What is the Impact Score?

The Impact Score shows how strongly a region is affected when oil supply is disrupted.

- 🔴 Red = High impact (strong economic effect)
- 🔵 Blue = Low impact (limited effect)
